# Start Here

This folder is a ready-to-go planning vault. To use it:

1. **Copy this whole `vault-template` folder** to wherever you want your planning files to live (e.g. `~/Documents/Planning`). Rename it whatever you like.
2. **Open it as a vault in Obsidian:** Open folder as vault → select your copy. (New to Obsidian? See `obsidian-setup.md` in the starter kit.)
3. **Install the Tasks plugin:** Settings → Community plugins → Browse → search "Tasks" (by Clare Macrae) → Install → Enable. Obsidian has to download plugins itself — they can't come pre-installed. This is the only one you need.
4. **Select the same folder as your Cowork workspace** and run the onboarding session (`onboarding-instructions.md` in the starter kit). It will fill in the skeleton files below with your projects and details.

## What's already here

- `+-2026-work-log.md` — your daily log, ready for entries (rename the year when a new one starts and archive the old file)
- `+-projects.md` — your project view, with one placeholder section to replace
- `+-projects-context.md` — Claude's reference file, same placeholder
- `+-Tasks.md` — the tasks dashboard; works as soon as the Tasks plugin is enabled
- `archive/` — where old work-log months and spent files go

Delete this file once you're set up.
